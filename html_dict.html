Dicionário de TAGS HTML

  • HTML: Elemento raiz de um documento HTML; envolve todo o conteúdo da página.
  • HEAD: Contém metadados e links para recursos externos, como título da aba da página, arquivo CSS e fontes.
  • META: Define metadados sobre o documento, como codificação de caracteres e configurações de viewport.
  • LINK: Relaciona o documento a recursos externos, como arquivos de estilo, ícones e fontes.
  • TITLE: Define o título da página exibido na aba do navegador.
  • BODY: Contém todo o conteúdo visível da página web.
  • HEADER: Seção de cabeçalho da página ou de uma seção, geralmente com logotipo, navegação ou título.
  • IMG: Tag para colocar uma imagem na página.
  • NAV: Define uma área de navegação com links para outras seções ou páginas.
  • UL: Cria uma lista não ordenada de itens.
  • LI: Define um item dentro de uma lista (ul ou ol).
  • A: Cria um hiperlink para outra página, seção ou recurso.
  • DIV: Elemento genérico de bloco usado para agrupar outros elementos e aplicar estilos.
  • ASIDE: Conteúdo lateral ou complementar ao conteúdo principal.
  • H3: Título de terceiro nível, usado para subtítulos e organização do conteúdo.
  • P: Define um parágrafo de texto.
  • HR: Insere uma linha horizontal para separar conteúdos.
  • STRONG: Tag semântica para destacar o texto com ênfase (aparece em negrito)
  • MAIN: Tag semântica para definir o conteúdo principal da página.
  • H1: Título de primeiro nível, geralmente o título principal da página.
  • SECTION: Tag semântica para definir uma seção temática do conteúdo.
  • H2: Título de segundo nível, usado para subtítulos.
  • FOOTER: Tag semântica para definir o rodapé da página ou de uma seção.
  • ADRESS: Fornece informações de contato, como endereço, e-mail ou telefone.
